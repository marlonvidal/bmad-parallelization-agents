# Version Information

## Current Version

**Version**: 1.0  
**Release Date**: February 2026  
**BMAD Compatibility**: v4.x

## Version History

### v1.0 (Current)
- Initial release
- Compatible with BMAD v4.x
- Supports parallel Frontend/Backend development
- Includes 5 custom agents: architect, pm, po, developer-front, developer-back
- One-command installation system
- Automatic backup functionality

## Roadmap

### v2.0 (Planned)
- **BMAD v6.x Compatibility**: Full support for BMAD version 6.x
- Updated agent definitions for v6.x architecture
- Enhanced features based on v6.x capabilities
- Backward compatibility considerations
